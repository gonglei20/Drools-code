### 1.规则引擎Drools介绍
   > Drools是基于java的一个开源规则引擎，它是由JBoss和红帽公司扩展支持,并实现Rete模式匹配算法的一个开源项目，实现了将业务决策从应用程序中分离出来。  
     Drools优点：  1、简化系统架构，优化应用  
　　　　2、提高系统的可维护性和维护成本  
　　　　3、方便系统的整合  
　　　　4、减少编写“硬代码”业务规则的成本和风险
